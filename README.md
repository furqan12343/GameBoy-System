# Gameboy System 🎮

Welcome to the **Gameboy System** repository! This project is a desktop GUI game system that integrates three classic games—**Wordle**, **Snake**, and **Hangman**—into a single, seamless experience using modern C++ (OOP concepts) and the SFML library for graphics and sound.

---

## 🚀 Features

✅ **Three Games in One:**  
- **Wordle** — Guess the hidden word in a limited number of attempts.
- **Snake** — Navigate your snake and collect items without hitting the walls or yourself.
- **Hangman** — Guess letters to avoid the hangman.

✅ **Modular & Object-Oriented Design:**  
- Each game is implemented as a separate, modular class using Object-Oriented Programming (OOP) principles.  
- Common components like the **sound system**, **leaderboard**, and **UI system** are shared and reusable across all games.

✅ **Graphical User Interface:**  
- Built using the **SFML library**, providing a polished and interactive GUI experience.  
- Smooth transitions between games with a consistent design language.

✅ **Sound System:**  
- Integrated **SoundSystem** class handles game sounds, background music, and effects for an immersive experience.

✅ **Leaderboard:**  
- Tracks and displays high scores across all games, enhancing replayability.

---

## 🛠️ Tech Stack

- **Language:** C++
- **Graphics & Audio:** [SFML](https://www.sfml-dev.org/) (Simple and Fast Multimedia Library)
- **Design Principles:**  
  - Object-Oriented Programming (OOP)  
  - Modularity  
  - Clear separation of concerns between game logic, graphics, and utilities
